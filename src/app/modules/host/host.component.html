It is important to set rules so that you and your guests are comfortable with each other right from the start

Some suggestions for Hosts:

  • Make indoor smoking rules.
  • Mention how your guest should cooperate in home security.
  • Mention kitchen usage rules.
  • Mention if you have a pet or will allow a pet.

Some suggestions for guests:

  • Take permission from the host if you are planning to have guests over.
  • Have respect for the host and the home, treat it as your own.
